# 吉他谱阅读器（平板优先 / 本地导入 / 离线曲库）

这是一个纯前端的吉他谱阅读器原型：

- 支持本地导入：TXT / TAB / ChordPro / Markdown / PDF / 图片
- 离线曲库：导入后保存在浏览器的 IndexedDB（只在当前设备当前浏览器生效）
- 平板适配：横屏默认双栏（曲库 + 阅读区），竖屏/小屏为抽屉曲库
- 阅读体验：字体大小调节、阅读模式、一键自动滚动

## 运行

这是静态站点，用任意本地静态服务器打开即可。

例如（Windows）：

```powershell
cd D:\codex\guitar-tab-viewer
python -m http.server 5173
```

然后用浏览器打开 `http://localhost:5173/`。
